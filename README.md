# Everything Indexer for Linux

A powerful file search tool for Linux, inspired by Windows Everything, with a Matrix-style theme.

![Matrix Theme](icon.png)

## Features

- **Lightning-fast file search** - Instant results as you type
- **Matrix-style UI** - Green-on-black theme with transparency
- **Powerful search syntax** - Supports wildcards, exclusions, and OR searches
- **File operations** - Rename (F2) and Delete (Del) files directly
- **Database indexing** - Fast searches by indexing drives/folders
- **Export results** - Save search results as CSV
- **Folder exclusion** - Exclude specific folders from indexing
